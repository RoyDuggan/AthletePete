# Beta Announcement — v0.1.0-beta (Brands Hatch Testing)

Release: https://github.com/RoyDuggan/VirtualPete/releases/tag/v0.1.0-beta

---

## Email / long form

**Subject:** Virtual Pete — Beta v0.1.0 ready for testing (Brands Hatch)

Hi team,

The first beta of **Virtual Pete** is ready to test ahead of Brands Hatch.

**What it does:** upload an AiM telemetry CSV, pick the laps to compare, and get
per-corner coaching with track minimaps, friction-circle (g-g) plots, and
speed-vs-distance traces.

**Please try:**
- Upload a session CSV and compare a few different lap pairs
- Switch the **Zone detection basis** (fastest lap / within 2% of fastest / all laps)
- Check the per-zone minimaps, g-g plots, and speed charts for sanity vs what you
  saw on track

**Release & notes:** https://github.com/RoyDuggan/VirtualPete/releases/tag/v0.1.0-beta

**Heads-up (beta):** needs an AiM CSV with beacon markers + GPS channels; setup
advisory is a placeholder; runs locally (dev build, not hosted yet).

Please report anything odd — wrong/missing data, crashes, confusing UI — ideally
**with the CSV that caused it**.

Thanks!
